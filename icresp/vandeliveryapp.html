Van Delivery System

Van delivery system suitable for any distributors to manage their daily transactions. System includes tablet based android application for Marketing executives and a web based ERP for managing business operations.Best part of our system is offline working mode of Android application, which will synchronize daily transactions with ERP while day close.

Android Application

Van delivery executive application has different modules like Inventory, Quotation, sale, sales return and finance. Executive can register a day after selecting his Route and group. Day register will load all required data to application from ERP. Daily operations like Sales, Receipts, Sales return, Quotation etc can be performed in Offline mode. Invoice printing can be done through bluetooth/wifi printers.System forces executives to meet all assigned customers daily. 'Day Close' will be allowed only after visiting all shops in the assigned route and day. Executive has to mark 'No Sale' with reasons in case of no transactions from any customer. Managers can see the executive performance reports through ERP. Executives can monitor targets, daily sales, daly collections, van stock values etc. in Application dashboard.

Web based ERP

VAT/GST enabled Web based ERP built on strong accounting platform allows any distributors to manage their accounts accurately. All Inventroy and Account modules are available in the system with user friendly interfaces. Multi-user system can create different user roles with specific access controls. System generates various reports like accounts reports, sales reports, purchase reports, executive reports, vat/gst reports etc which helps decision making process effectively.
